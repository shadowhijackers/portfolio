# Run Project

Whenever you make changes to the application that require verification, start the development server.

## Command

```bash
npm run dev
```

## Guidelines

- Run the command from the project's root directory.
- If the development server is already running, do not start another instance.
- Wait for the server to compile successfully.
- Check the terminal for build or runtime errors.
- If errors occur, fix them before continuing.
- Verify that the application is accessible in the browser.
- Do not terminate the development server unless explicitly requested.
